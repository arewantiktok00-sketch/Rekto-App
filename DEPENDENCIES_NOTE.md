# Required Dependencies for Link-in-Bio Feature

Please install these packages:

```bash
npm install @gorhom/bottom-sheet react-native-image-picker expo-image-picker react-native-draggable-flatlist
```

Or with Expo:

```bash
npx expo install @gorhom/bottom-sheet expo-image-picker react-native-draggable-flatlist
```

Note: `react-native-draggable-flatlist` requires `react-native-gesture-handler` and `react-native-reanimated` which are already installed.
